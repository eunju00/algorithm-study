# Problem 10171

## 구현

### 문제 링크
<https://www.acmicpc.net/problem/10171>

### 어려웠던점
1. \와 '를 사용하는 이모지라서 오류가 발생하여 해결하기 힘들었다.
2. '는 뒤에 \를 붙이면 오류가 나고, 앞에 붙여야 오류가 나지 않았다.

### solved
1. \이면 \\라고 입력하면 \만 출력된다.
2. '는 앞에 \를 붙여넣으면 오류가 없다.

### 주의할점
1. '는 뒤가 아니라 꼭 앞에 \를 붙여야 오류가 없었다.

### code
```javascript
console.log('\\    /\\');
console.log(' )  ( \')');
console.log('(  /  )');
console.log(' \\(__)|');
